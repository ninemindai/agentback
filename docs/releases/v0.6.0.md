# v0.6.0

The actor-model release. AgentBack gains a Zod-typed **actor runtime**, the
schema-explorer learns to **export OKF knowledge bundles**, and the CLI gets a
**secretless Cloudflare deploy**.

## ‚ú® Highlights

### Actor runtime ‚Äî `@agentback/actors` + `@agentback/actors-redis`

`@actor` service classes with `@actorCommand` methods compile (at `start()`)
into a transport-neutral runtime: a stable `{type, id}` address, per-identity
serialized turns, idempotent `requestId` replay, and rollback on a
failed/schema-invalid turn. Callers invoke through `ACTOR_REGISTRY` ‚Äî never the
actor instance directly. `InMemoryActorsComponent` is the single-process
reference adapter; `actors-redis` coordinates one turn at a time across
processes via a per-identity lease (state + dedup result commit atomically in
Lua ‚Äî the lease token is the sole mutual-exclusion guard).

- **Event-sourced runtime** ‚Äî a command turn may also return `events` (domain
  facts); `EventSourcedActorsComponent` persists them to a per-identity
  append-only log atomically with state. Read via `registry.events(type, id)` /
  `registry.subscribe(handler)`. State stays authoritative ‚Äî "state + event
  log", not full event sourcing.
- **`@actorQuery`** ‚Äî read-only methods run lease-free (no turn) and join the
  typed proxy.
- **`@injectActor`** ‚Äî inject a typed actor accessor, no client class needed.
- **Typed proxy** ‚Äî `registry.ref(ActorClass, id).command(input, {requestId})`
  (pass the class, not the name).

State is an explicit method arg/return, never an instance field. See
[`examples/hello-actors`](../../examples/hello-actors) and
[docs/actor-model.md](../actor-model.md).

### OKF export ‚Äî `@agentback/schema-explorer`

The schema-explorer's provenance graph serializes to an
[OKF](https://github.com/GoogleCloudPlatform/knowledge-catalog/tree/main/okf)
(Open Knowledge Format) bundle ‚Äî a portable, vendor-neutral, git-versionable
directory of markdown + YAML-frontmatter docs an agent can ingest verbatim. It's
a sixth, comprehension-oriented projection of the one Zod source of truth
(alongside OpenAPI, MCP `inputSchema`, REST routes, and Drizzle tables).

The bundle is two-tier and cross-linked (`schemas/` + `surfaces/` with auto
`index.md` files); Drizzle-backed entities are typed `table`, others
`reference`; the framework's own dev-tooling controllers are omitted by default
so a bundle describes the app. Browse and download it (`.zip` / single `.md`)
from the new **Knowledge** tab, fetch `GET /schema-explorer/api/okf`, or call
`buildOkfBundle(app)` / `inventoryToOkf(inv, {exclude?})` (emit-only,
deterministic).

### Secretless Cloudflare deploy ‚Äî `@agentback/cli`

`agentback deploy cloudflare --temporary` deploys to a throwaway Cloudflare
preview account ‚Äî no signup, account, or API token. Wrangler provisions a
temporary account on the fly, deploys, and prints a claim URL; the deployment
expires after 60 minutes unless claimed. It's the secretless way to run a *real*
deploy in CI (where `--dry-run` only proves the bundle is clean). `--temporary`
only works **unauthenticated** (the inverse of the normal `wrangler login`
prerequisite); requires wrangler ‚â• 4.102. See
[docs/guides/deploy-to-edge.md](../guides/deploy-to-edge.md).
